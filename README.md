# 🍺 DrunkStat

A party game where sports knowledge meets cold drinks. Test your stat IQ against friends in this fast-paced trivia showdown.

## 🎮 How to Play

1. **Host a Game**: Select your sports categories and stat types
2. **Share the Code**: Players join using the game code
3. **Answer Fast**: 30 seconds per question - no time to Google!
4. **See Who Knows Best**: Live scoreboard shows results after each round

## 🏆 Features

- **5 Sports Categories**: College Football, NFL, College Basketball, NBA, MLB
- **3 Stat Types**: Individual Stats, Fantasy Points, Team Stats
- **Real-time Gameplay**: Everyone plays simultaneously
- **Live Scoreboard**: See who got it right instantly
